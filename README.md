# FileFit
Programa para testar combinações de pastas e arquivos para encher um DVD sem desperdício de espaço.

Foi feito em 2011, antes de eu aprender a usar threads e acumuladores.

O algoritmo usado é bem simples e ineficiente (aka testar todas as combinações possíveis e imagináveis).

Com 8GB de memória dá estouro com pouco mais de 27 pastas.

Enfim, qualquer dia eu arrumo isso. Ou não.
